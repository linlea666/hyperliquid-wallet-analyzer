# 宝塔面板快速部署（5分钟搞定）

## 🚀 超简单部署步骤

### 步骤 1: 上传项目（1分钟）

1. 宝塔面板 → 「文件」→ `/www/wwwroot/`
2. 上传 `gendan-deploy-*.zip` 并解压
3. 重命名为 `gendan`

### 步骤 2: 添加 Python 项目（2分钟）

1. 「软件商店」→ 安装「Python项目管理器」（如未安装）
2. 「Python项目管理器」→ 「添加项目」

**填写信息**：
```
项目名称: hyperliquid-backend
Python环境: 3.11.13（或更高）
启动方式: 命令行启动
项目路径: /www/wwwroot/gendan/backend
启动命令: uvicorn app.main:app --host 0.0.0.0 --port 8000
通讯协议: asgi
端口: 8000
安装依赖包: /www/wwwroot/gendan/backend/requirements.txt
```

3. 点击「确定」
4. 点击「模块」→ 「批量安装」→ 选择 `requirements.txt`
5. 点击「启动」✅

### 步骤 3: 构建前端（1分钟）

1. 「文件」→ 进入 `/www/wwwroot/gendan/frontend`
2. 点击「终端」
3. 运行：
```bash
npm config set registry https://registry.npmmirror.com
npm install
npm run build
```

### 步骤 4: 创建网站（1分钟）

1. 「网站」→ 「添加站点」
2. 域名：您的域名或 IP
3. 根目录：`/www/wwwroot/gendan/frontend/dist`
4. PHP：纯静态
5. 点击「提交」

### 步骤 5: 配置反向代理（1分钟）

1. 网站「设置」→ 「反向代理」→ 「添加反向代理」
2. 代理名称：`api`
3. 目标URL：`http://127.0.0.1:8000`
4. 代理目录：`/api`
5. 点击「提交」

6. 网站「设置」→ 「配置文件」
7. 在 `location /` 中添加：
```nginx
try_files $uri $uri/ /index.html;
```
8. 保存并重启 Nginx
