# 🚀 快速启动指南

## 一键启动（macOS）

### 方法一：使用启动脚本（推荐）

```bash
cd /Users/huahua/Documents/gendan
./start.sh
```

### 方法二：手动启动

**1. 启动后端**
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

**2. 启动前端**（新开一个终端）
```bash
cd frontend
npm install
npm run dev
```

## 访问系统

- 🌐 **前端界面**：http://localhost:5173
- 📊 **后端 API**：http://localhost:8000
- 📚 **API 文档**：http://localhost:8000/docs

## 一键停止

```bash
./stop.sh
```

## 首次使用

1. **导入钱包**：
   - 点击"钱包列表" → "导入钱包"
   - 输入钱包地址（支持批量，每行一个）
   - 点击"导入"

2. **查看数据**：
   - 在钱包列表中点击"查看详情"
   - 查看各项指标和交易历史

3. **查看看板**：
   - 点击导航栏"看板"
   - 查看整体统计和异动信息

## ⚠️ 重要提示

**当前版本使用 Mock 数据**，要使用真实数据需要：

1. 查看 HyperLiquid API 文档
2. 编辑 `backend/app/services/hyperliquid.py`
3. 替换 `get_wallet_data()` 方法中的 Mock 代码为真实 API 调用

## 📝 日志位置

- 后端日志：`logs/backend.log`
- 前端日志：`logs/frontend.log`
- 应用日志：`backend/logs/app.log`
- 错误日志：`backend/logs/error.log`

## 🆘 遇到问题？

1. 检查 Python 和 Node.js 版本
2. 查看日志文件
3. 确认端口 8000 和 5173 未被占用
4. 重新运行 `./start.sh`
