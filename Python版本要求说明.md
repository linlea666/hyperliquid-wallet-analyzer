# Python 版本要求说明

## ✅ Python 版本要求

### 最低版本要求

- **最低版本**：Python 3.8+
- **推荐版本**：Python 3.9+ 或 3.10+
- **Python 3.11.13**：✅ **完全支持，推荐使用！**

## 🔍 依赖包版本要求

项目使用的依赖包都支持 Python 3.8+：

- FastAPI 0.104.1：支持 Python 3.8+
- uvicorn 0.24.0：支持 Python 3.8+
- pandas 2.1.3：支持 Python 3.8+
- numpy 1.26.2：支持 Python 3.8+
- pydantic 2.5.0：支持 Python 3.8+

## 💡 宝塔面板配置建议

### 推荐配置

在宝塔面板「Python项目管理器」中：

- **Python版本**：选择 `3.11.13` ✅ **推荐**
- 或选择 `3.10.x`（如果 3.11 不可用）
- 或选择 `3.9.x`（备选）

### 配置步骤

1. 进入「Python项目」→「添加Python项目」
2. **Python版本**：选择 `3.11.13`（如果可用）
3. 其他配置按正常流程
