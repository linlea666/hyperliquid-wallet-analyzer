## 📝 日常更新（3 步）

### 在 Cursor 中：

1. **修改代码**
   - 在 Cursor 中编辑文件

2. **提交代码**
   - 点击左侧「Source Control」图标（分支图标）
   - 在「Message」输入提交说明
   - 点击「Commit」

3. **推送到 GitHub**
   - 点击「Sync Changes」按钮
   - 完成！✅

## 🎯 快捷键

- **打开 Source Control**：`Cmd+Shift+G`
- **提交**：`Cmd+Enter`（在 Source Control 面板）
- **查看差异**：点击修改的文件

---

## 📋 常用操作

### 查看修改

点击「Source Control」图标，查看：
- 绿色 `M`：已修改
- 绿色 `A`：新添加
- 红色 `D`：已删除

### 查看文件差异

点击修改的文件，会显示修改前后的对比。

### 撤销修改

- 右键文件 → 「Discard Changes」

### 查看提交历史

- 右键文件 → 「Open Timeline」
- 或命令面板：`Git: View History`
