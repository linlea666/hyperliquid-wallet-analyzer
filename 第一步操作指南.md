# 🎯 第一步操作指南（小白版）

## 📋 您现在需要做什么？

### ✅ 第一步：在 GitHub 创建仓库

#### 1. 打开 GitHub

在浏览器打开：https://github.com

#### 2. 登录或注册

- 如果没有账号：点击「Sign up」注册
- 如果有账号：点击「Sign in」登录

#### 3. 创建新仓库

1. **点击右上角的「+」号**（绿色按钮旁边）
2. **点击「New repository」**（新建仓库）
   - ⚠️ 选择「新建仓库」，不是「导入仓库」

#### 4. 填写信息

**Repository name（仓库名称）**：
```
hyperliquid-wallet-analyzer
```

**Description（描述）**（可选）：
```
HyperLiquid 钱包分析系统
```

**Public 还是 Private？**

**推荐选择 Public（公开）** ✅：
- ✅ 最简单，无需配置认证
- ✅ 服务器拉取代码最简单
- ✅ 适合学习项目

**如果选择 Private（私有）**：
- 🔒 代码只有您能看到
- ⚠️ 服务器拉取需要配置 Token（稍复杂）

**其他选项**：
- ⚠️ **不要**勾选任何选项（README、.gitignore、license 都不要）

#### 5. 创建

点击绿色的「Create repository」按钮

---

### ✅ 第二步：复制仓库地址

创建成功后，GitHub 会显示仓库地址，类似：

```
https://github.com/YOUR_USERNAME/hyperliquid-wallet-analyzer.git
```

**复制这个地址**，稍后会用到。

---

### ✅ 第三步：在本地终端操作

打开终端（Terminal），依次运行：

```bash
# 1. 进入项目目录
cd /Users/huahua/Documents/gendan

# 2. 初始化 Git 仓库
chmod +x 初始化Git仓库.sh
./初始化Git仓库.sh

# 3. 连接 GitHub（替换 YOUR_USERNAME 为您的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/hyperliquid-wallet-analyzer.git

# 4. 推送代码
git branch -M main
git push -u origin main
```

**如果选择 Public 仓库**：
- 提示输入用户名：输入您的 GitHub 用户名
- 提示输入密码：**直接回车**（Public 仓库不需要密码）

**如果选择 Private 仓库**：
- 提示输入用户名：输入您的 GitHub 用户名
- 提示输入密码：输入 Personal Access Token（需要先生成，见下方）

---

### ✅ 第四步：验证

刷新 GitHub 页面，应该能看到所有项目文件了！

---

## 🔐 Private 仓库需要 Token（如果选择私有）

### 生成 Token 步骤：

1. GitHub → 右上角头像 → **Settings**
2. 左侧菜单 → **Developer settings**
2. → **Personal access tokens** → **Tokens (classic)**
3. → **Generate new token (classic)**
4. 填写：
   - **Note**: `服务器部署`
   - **Select scopes**: 勾选 `repo`
5. → **Generate token**
6. **⚠️ 复制 Token**（只显示一次，要保存好）

### 使用 Token：

推送代码时，密码输入框粘贴 Token（不是登录密码）

**小白推荐选择 Public（公开）**：
- ✅ 最简单
- ✅ 服务器拉取代码无需配置

**如果代码敏感，选择 Private**：
- 🔒 更安全
- ⚠️ 需要配置 Token（一次配置，后续简单）
